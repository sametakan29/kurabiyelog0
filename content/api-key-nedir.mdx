# API Key Nedir ve Ne İşe Yarar?

**API Key (API Anahtarı)**, bir uygulamanın başka bir servise (örneğin Google Maps veya Gemini) kendini tanıtmak için kullandığı özel bir şifredir. **"API Key nedir"** diye merak edenler için kısaca dijital kapı anahtarı diyebiliriz.

### Neden Önemli?
Eğer API anahtarınızı GitHub'a yüklerseniz, başkaları sizin adınıza harcama yapabilir!

```javascript
// Yanlış kullanım!
const apiKey = "12345-gizli-anahtar"; 
```

Bunun yerine .env dosyası kullanmalısınız.
