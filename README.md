# Tech Blog

A personal tech news and blog site built with Next.js. Posts cover technology-related topics in a news-style format.

## Requirements

- Node.js 18+
- npm, yarn, pnpm, or bun

## Build & Run

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

For production:

```bash
npm run build
npm start
```

## Notes

- Built with Next.js App Router.
- Deployed easily via [Vercel](https://vercel.com).
