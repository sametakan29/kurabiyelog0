# API Key Nedir?

API Key, bir uygulamanın başka bir servise (örneğin Google Maps veya Gemini) kendini tanıtmak için kullandığı özel bir şifredir.

### Neden Önemli?
Eğer API anahtarınızı GitHub'a yüklerseniz, başkaları sizin adınıza harcama yapabilir!

```javascript
// Yanlış kullanım!
const apiKey = "12345-gizli-anahtar"; 
```

Bunun yerine .env dosyası kullanmalısınız.
